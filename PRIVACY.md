# Privacy Policy — Smart Filter Bar

**Last updated:** March 12, 2026

## Overview

Smart Filter Bar is a Power BI custom visual that provides a horizontal filter bar with dropdowns, badges, sorting, dark mode, and cross-filtering capabilities.

## Data Collection

Smart Filter Bar **does not collect, store, or transmit any personal data**. All data processing occurs entirely within your Power BI environment.

## What This Visual Does

- Reads data from the Power BI data model to display filter options
- Applies filters and cross-filtering within the Power BI report
- Stores user preferences (such as dark mode) locally within the Power BI session

## What This Visual Does NOT Do

- Does not send data to external servers
- Does not collect analytics or telemetry
- Does not use cookies or tracking technologies
- Does not access data outside of the Power BI report context
- Does not store any data permanently

## Third-Party Services

This visual does not integrate with or send data to any third-party services.
