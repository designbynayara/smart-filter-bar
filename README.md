# Smart Filter Bar

**Smart Filter Bar** is a certified Power BI custom visual that replaces the default slicer with a modern horizontal filter bar featuring dropdowns, badges, sort, dark mode, and cross-filtering.

## Features

- Horizontal filter bar with dropdowns
- Active filter badges
- Sort control
- Dark mode
- Cross-filtering support
- Multi-filter panel
- Vertical mode
- Highly customizable (colors, fonts, border radius, and more)

## Installation

1. Download the latest `.pbiviz` file from [Releases](https://github.com/designbynayara/smart-filter-bar/releases)
2. In Power BI Desktop, go to **Visualizations > Import a visual from a file**
3. Select the downloaded `.pbiviz` file

## Usage

1. Add the visual to your report canvas
2. Drag fields to the **Category** data role
3. Customize colors, fonts, and behavior in the **Format** panel
